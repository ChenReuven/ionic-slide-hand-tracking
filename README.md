Hand Detection + Ionic + RevealJS
=====================

A starting project for RevealJS With Ionic And Hand Detection For move slide left, right, up and down.

## Using this project
u must install first ionic

```bash
$ npm install -g ionic
```

Then run:

For Web:
```bash
$ ionic serve
```
For ios:
```bash
$ ionic run ios
```

For android:
```bash
$ ionic run android
```

Note: U must configure your ionic ios and android before u run those command. [See Here] (http://ionicframework.com/docs/guide/installation.html)
